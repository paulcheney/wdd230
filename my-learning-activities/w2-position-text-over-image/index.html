Responsive Image with Overlay Text

Lesson Goals

  • Add text over the image using CSS Position

Overview

As shown in the examples below from Mastercard and Lewis State University, there are times when you want to add live text or even clickable buttons over an image.

This can be easily done using CSS positioning. This same technique can be used over a hero video clip.

ACTIVITY: Overlay text on the image

For this activity to work, you need to create a parent tag with two children. For this example, lets use a division tag as the parent element. Then inside that we will add an image as the first child and a span as the second child.

Example 

<div class="hero">

  <img src="images/riverwalk.webp" alt="riverwalk" width="960" height="384">

  <span>Downtown River Walk</span>

</div>

Yes, it's really that easy.

Now it's time for the CSS. We need rules for the parent element and rules for each of the two children elements. Based on using a class name of 'hero' you should start with the following css.

Example 

.hero {
}
.hero span {
}
.hero img {
}

Currently the text is displayed immediatly below the photo. Lets fix that problem next.

The parent division element needs to have its position set to relative.

This allows us to set the position of the span child to absolute. For now, set the bottom to 0 and the left to 0.

If you're following along, you will now see the text over the bottom edge of the photo.

Example 

  .hero {
    position: relative; 
  }
  .hero img {
  }
  .hero span {
    position: absolute;
    bottom: 0;
    left: 0;
  }

An image should be displayed with the width at 100% of it's parent tag and the height should be set to auto. This makes the image scale as the page changes width for different devices.

Example 

.hero img {
  width: 100%;
  height: auto;   
}

There is another problem we need to solve but in order to see it we need to add a border around out span tag like this.

border: dashed red 1px;

With the border added we can see that the dashed line is hanging over the bottom of the image. This is because by default, an image is an inline tag.

We can solve this problem by changing the img to display as a block. Now that we have solved that problem, we can remove the border.

Next we will add a transparenet white background behind the text using rgba() like this.

background-color: rgba(255,255,255,0.5);

If you explore the position tag, you will see bottom and left as well as right and top. Imagine how you could use these numbers to move this tet anywhere on the image.
